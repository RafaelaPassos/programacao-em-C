# Lista Duplamente Encadeada em C

## 📌 Descrição
A estrutura permite percorrer os elementos tanto do início para o fim quanto do fim para o início, por meio de ponteiros para o próximo e o elemento anterior.

---

## Funcionalidades
- Inserção de elementos
- Remção de elementos
- Percorrer a lista para frente
- Percorrer a lista para trás

---

## 🛠️ Tecnologias
- Linguagem C

---

## 🎯 Objetivo
Praticar o uso de ponteiros, alocação dinâmica de memória e estruturas de dados.
