# Insertion Sort em C

## 📌 Descrição
O Insertion Sort organiza os elementos do vetor inserindo cada item na posição correta dentro da parte já ordenada do vetor.

---

## 🛠️ Tecnologias 
- Linguagem C

---

## 🎯 Objetivo
Desenvolver habilidades em algortmos de ordenação e reforçar conceitos de lógica de programação.
